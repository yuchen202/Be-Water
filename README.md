## 🌟 Project Overview

**Be Water** is a large-scale computational social science project that explores the evolutionary fitness of different trading strategies in the volatile cryptocurrency market. By simulating a society of **10,000 heterogeneous agents** over 5 years of high-frequency (5m) data, this project provides a mathematical and empirical proof that **Trend-Following** is the most resilient phenotype for long-term survival.

### 🧪 Key Scientific Discoveries
- **The Momentum Singularity:** In a zero-friction Utopian environment with a Universal Basic Income (UBI) safety net, the population naturally converges from 5 archetypes to a 75%+ dominance of Trend-Followers.
- **The Negative Skew Trap:** Mean-reversion and Grid strategies, despite consistent micro-profits, are systematically purged during "fat-tailed" market shifts.
- **Positive-Sum Emergence:** A collective of adaptive agents can systematically extract value from non-stationary markets, growing aggregate social wealth by 10%+.

## 🏗️ System Architecture

The project consists of two core components:

1.  **MAS-Utopia Simulation Engine**: A high-performance, memory-optimized backtesting environment (SQLite + O(1) Matrix Lookup) that simulates 5 years of evolution in hours.
2.  **LLM Cognitive Prosthesis (Sim2Real)**: A real-world deployment system that translates simulation findings into actionable US/HK stock market signals using **DeepSeek-R1** Chain-of-Thought (CoT) reasoning.

### 📁 File Structure
- `run_backtest.py`: The "Time Machine" - high-speed evolutionary engine.
- `dashboard.py`: Streamlit-based "Observation Post" for real-time visualization.
- `agent_core.py`: Database schema and genetic encoding of trading archetypes.
- `jiqrhf.py` / `zxg.py` / `jksy.py`: Real-world LLM-driven market surveillance system.
- `Paper/`: LaTeX source for the associated scientific paper.

## 🚀 Getting Started

### Prerequisites
- Python 3.12+
- 16GB+ RAM (40GB recommended for full 10,000-agent simulation)

### Installation
```bash
git clone https://github.com/your-username/Be-Water.git
cd Be-Water
pip install -r requirements.txt
```

### Running the Simulation
1. **Start the Engine**:
   ```bash
   python run_backtest.py
   ```
2. **Launch the Dashboard**:
   ```bash
   streamlit run dashboard.py
   ```

## 📊 Visualizing Results
The dashboard provides several high-dimensional views:
- **Ecological Succession (River Plot)**: Watch the real-time rise and fall of trading species.
- **Lorenz Curve & Gini Coefficient**: Monitor the evolution of wealth inequality.
- **3D Trajectory**: Trace the compounding paths of the "Elite" class.
